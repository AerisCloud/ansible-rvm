RVM
===

This Ansible role installs RVM as root and the required ruby version

Requirements
------------
Tested with CentOS 6.5, Debian 6, 7, Fedora 18, 19, 20, Ubuntu 10.04, 12.04, 13.10.

Role Variables
--------------
- `rvm.ruby_version` per default set to `ruby-2.1.1`
